Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cres for the appearance of the document.

HTML elemamts are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded intonthe rendered page. It provides a means to create sturctured documents by denoting structual semantics for text such as headings, paragraphs, lists, links, quotes and other items.
